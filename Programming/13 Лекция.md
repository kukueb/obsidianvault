## Указатели
### *Указатели на объект*
Содержит адрес области памяти, в которой хранятся данные определённого типа.
Синтаксис указателя
`тип *имя;`
`int *a, b, *c;`

### *Указатель на void*
Применяется если конкретный тип объекта не определён

### Инициализация указателя
Для инициализации указателя можно использовать несколько способов.
`int* p{};`
1) С помощью операции получения адреса
```c++
int a = 5;
int *p = &a;
```
2)  С помощью проинициализированного указателя
```c++
int *r = p;
```
3) Присваивание в явном виде
```c++
char *cp = (char*)0xB8000000;
// где 0xB8000000 - шестнадцатиичная константа, (char*) - операция приведения типа.
```
### Получение адреса и оператор &
С помощью операции `&` можно получить адрес некоторого объекта, а затем присвоить его указателю
```c++
int number {25};
int *pnumber {&number};
```

```c++
int number {25};
auto *pnumber {&number}; // Указатель pnumber хранит адрес  переменной number
cout << "pnumber addr: " << &pnumber;
cout << "pnumber value: " << *pnumber;
```

```c++
int x = 10;

```
Используя обращения по адресу, можно поменять значение переменной, обращаясь не к ней, а к значению по адресу

### Операции с указателями
*Присваивание адреса*
```c++
int a {10};
int *pa {&a};
```
*Разыменование указателя*
```c++
int a {10};
int *pa {&a};
cout << *pa << endl; // 10
cout << a << endl; // 10
cout << pa << endl; // 0x...
```
*Присвоение указателя другому указателю*
```c++
int a {10};
int *pa {&a};
int *pb = {pa};
```

*Приведение типов*
За счёт операции приведения типов указатели разного типа, ссылающиеся на одну область памяти будут давать разные результаты
```c++
int a = 123;
int *pi = &a;
char *pc = (char*)&a;
float *pf = (float*)&a;
cout << pi << *pi << endl; // 66fd9c 123
cout << pc << *pc << endl; // 66fd9c {
cout << pf << *pf << endl; // 66fd9c 123
```

*Ссылки на указатели*
```c++
int a {10};
int b {6};
int *p{};
int *&pRef {p}; // ссылка на указатель
cout << "p value = " << *p << endl; // 10
*pRef = 70; // изменяем значение по адресу, на который указывает указатель
cout << "p value = " << *p << endl; // 70
pRef = &b 
cout << "p value = " << *p << endl; // 6
```
Так как ссылка не является объектом, то нельзя определить указатель на ссылку.
Однако, можно определить ссылку на указатель. Такая конструкция позволяет изменять адрес указателя
```c++
int a {10};
int *pa {&a};
cout << "address of pointer = " << &pa; // адрес указателя
cout << "address stored in pointer = " << pa; // адес, котоый хранится в указателе - адрес переменной a
cout << "value on pointer = " << *pa; 
```

*Операции сравнения*
К указателям можно применять операции сравнения, но только к указателям одного типа
```c++
int a {10};
int b {20};
int *pa {&a};
int *pb {&b};
if (pa > pb)
	cout << "pa (" << pa << ") is greater than pb (" << pb << ")" << endl; 
else
	cout << "pa (" << pa << ") is less or equal pb (" << pb << ")" << endl; 
```

*Арифметические операции*
Можно использовать сложение, вычитание, инкремент и декремент
В арифметических операциях учитывается размер объекта, на который ссылается указатель.
Например: "++" увеличивает объект на 1, а инкремент для указателя увеличивает значение адреса на размер типа указателя.
```c++
int n {10};
int *pn {&n};
cout << "address = " << pn << "\tvalue = " << *pn << endl; // 0x81315ffd84 10
pn++;
cout << "address = " << pn << "\tvalue = " << *pn << endl; // 0x81315ffd88 828374403
pn--;
cout << "address = " << pn << "\tvalue = " << *pn << endl; // 0x81315ffd84 10
```

*Вычитание указателей*
Результатом вычитания указателей является расстояние между ними

### Указатель на константу
```c++
const int a {10};
const int *pa {&a};
// *pa = 34; // так делать нельзя

int a {10};
const int *pa {&a};
a = 22 // Всё ок
// *pa = 34; // так делать нельзя
```

### Константный указатель
Константные указатели не могут изменять адрес, который в них храниться, но могут изменять значения по этому адресу.
```c++
int a {10};
int *const pa {&a};
*pa = 22;
int b = {45};
// pa = &b; // так делать нельзя
```

### Константный указатель на константу
не позволяет менять ни хранимый в нём адрес, ни значение по адресу
```c++
int a{10};
const int *const pa {&a};
// *pa = 22; // так делать нельзя
int b {45};
// pa = &b; // так делать нельзя
```

## Указатели и массивы
```c++
int nums[] {1,2,3,4,5};
cout << "nums[0] address " << nums << endl;
cout << "nums[0] value " << *nums << endl;
```
