## Введение в строки
Для работы со строками необходимо подключить класс `string`. Объект этого класса содержит последовательность символов типа `char`.
```c++
#include <string>
using namespace std;

int main() {
	string message {"Hello"};
	cout << "Message: " << message << endl;
}
```

```c++
string hello {"Hello"};
char c {hello[1]}; // 'e'
hello[0] = "M";
std::cout << hello << std::endl; // Mello
```

Строка в отличии от массива может обрабатываться как **по-символьно** так и **целиком**.
```c++
for (const char c: message) {
	if (c == 'l') {
		count++;
	}
}
```
Функция `getline` выполняет ввод до нажатия `Enter`. Первый параметр - **поток ввода**, второй параметр - **переменная** в которую выполняется ввод.

## Функции
Функции определяют набор действий и назначают ему имя.
Такой набор можно вызывать многократно.
Определение функции имеет вид:
```c++
[класс] тип имя ([ список_формальных_параметров ])
{
	тело функции
}
```

Для возвращения результата функции в теле функции указывается оператор `return`.
Если тип функции любой кроме `void`, то тип возвращаемого результата должен совпадать с типом функции.
Функция без результата - **процедура**

Обычно вызов функции происходит после её определения, но если это не так, то необходимо функцию объявить.
Объявление функции называется **прототипом**
```c++ 
[ класс ] тип имя ([ список_формальных_параметров ]);
```

```c++
void hello();

int main() {
	hello();
	hello();
}

void hello() {
	cout << "hello" << endl;
}
```

### Класс
Модификатор **класс** может принимать 2 значения:
- `extern` - глобальная видимость во всех модулях программы (по умолчанию);
- `static` - видимость только в пределах модуля, в котором определена функция.

*Тип возвращаемого функцией* значения
1) `return`*`выражение`*`;`
2) `return;`
Функция вызывается с помощью указания имени и списка фактических параметров.
Вызов функции:
```c++
имя ([ список_фактических_параметров ])
```
Списки **формальных** и **фактических** параметров должны совпадать по *количеству*, *порядку* и *типам*.

### Прототип функции
Пример 1. Использование прототипов функций. Пример без прототипа
```c++
double cude (double a) {
	return a*a*a;
}

int main (void) {
	double pi = 3.1415;
	cout << cube(pi);
	return 0;
}
```
Пример с прототипом
```c++
double cube(double);

int main(void) {
	double pi = 3.1415;
	cout << cube(pi);
	return 0;
}

double cube (double a) {
	return a*a*a;
}
```
