## Оператор присваивания
Выполняется справа налево
Для явного преобразования при присваивании используется конструкция
`<имя_переменной_1> = (тип_данных)<имя_переменной_2>`
Например:
`arg_long = (long)arg_f`

В формате C++20 допустимо:
```
int a{}, b{}, c{};
a = b = c = 34;
```
Существуют операторы сложного присваивания

```
+=; 
-=;

*=;
/=;
%=;

<<=;
>>=;
&=;
|=;
^=; Присваивание после операции исключающего ИЛИ
```
## Блок и составной оператор
**Составной оператор** содержит несколько операторов и, если в нём определены переменные, то он называется блоком.
```
{
	n++;           // это составной оператор
	summa+=n;
}

{
	int n=0;
	n++;           // это блок
	summa += n;
}
```

## Средства ввода-вывода
В языке C используется следующие операторы ввода-вывода
1) `printf( форматная строка, список аргументов);` форматная строка - строка символов, заключенных в кавычки, которая показывает, как должны быть напечатаны аргументы, например: ` print("Значение числа Пи равно %f\n", pi);`
2) `scanf( форматная строка, список аргументов);` В качестве аргументов используются адреса переменных, например: `scanf(" %d%f ", &x, &y);`
В языке C++
3) << - операция записи данных в поток
4) >> - операция чтения данных из потока
Существует два потока: ввод и вывод
cin и cout, доступные в библиотеке std;

## Условный оператор
### Синтаксис
В полной форме
```
if (выражение B)
	оператор_S1;
else
	оператор_S2;
```
В краткой форме
```
if (выражение B)
	оператор_S1;
```
### Семантика
полная форма: Если вычисленное выражение B не равно 0, то выполняется оператор S1, если результат равен 0 выполняется оператор S2

Краткая: Если вычисленное выражение B не равно 0, то выполняется оператор S1, если результат равен 0, то выполняется следующий оператор.

### Примеры
```
if (x > 0)
{
	x = -x;
	cout << x;
}
```

```
int a; bool h;
cin >> a;
if (a) h = true;
else h = false;
cout << h;
```
### Вложенность условных операторов
Так как условный оператор является оператором, то он может находиться в составном операторе другого условного оператора.
```
if (a>b) 
	if (a>c) d=a;
	else d=c;
else if (b>c) 
	d=b;
	else d=c;
```

Общий вид
```
if (B1) if (B2) S1; else S2;
```
ИЛИ
```
if (B1) 
	{ if (B2) S1; }
else S2;
```

### Сложные условия
&& - логическое И
|| - логическое ИЛИ
! - логическое Нет
`if (4 < 6 && (5 > 6 || 5 < 6))`
При необходимости изменения порядка проверки следует операции с более низким приоритетом поместить в круглые скобки:
`if (4<6 && (5>6 || 5<6))`

## Оператор выбора
### Синтаксис
```c++
switch (выражение)
{
	case константа1: <операторы>
	case константа2: <операторы>
	...
	[default: <операторы>] // необязательный блок
}
```
Выражение должно быть скалярного дискретного типа.

### Семантика
Вычисляется выражение и его значение соотносится с константами веток **case**
При первом совпадении выполняются операторы **этой ветки** и **все** последующие.
Если значение выражения не совпало **ни с одной** константой, то выполняется ветка **default** при её наличии.
```
#include <iostream>

void main()
{
	int i;
	cout << "\nEnter the number";
	cin >> i;
	switch (i) 
	{
		case 1: cout << "\nThe number is one";
		case 2: cout << "\n2*2 = " << i*i;
		case 3: cout << "\n3*3 = " << i*i; break;
		case 4: cout << "\n" << i << " is very beautiful";
		default: cout << "\nThe end of work";
	}
}
```
## Циклы
Существует 3 базовых цикла:
 - цикл с предусловием
 - с постусловием
 - с параметром
### Цикл с предусловием
#### Синтаксис
```
while (<условие>) 
{
	<тело цикла>
}
```
#### Семантика
Если вычисленное условие не равно нулю, то выполняются операторы в теле цикла, затем снова вычисляется условие и если результат условия равен нулю, то выполняется следующий за циклом оператор.
Для того чтобы не создавалась операция зацикливания, необходимо, но не достаточно, чтобы внутри тела цикла присутствовал оператор, изменяющий условие.
#### Примеры
`while a>0 {c++;}` - вечный цикл
**Вычисление $n!$**
```c++
int n, f=1, i=0;
cin >> n;
while (i<=n)
{
	i++;
	f*=i;
}
cout << f;
```
##### Схема обработки цифр в числе 
```c++
int N;
cin >> N;
while (N > 0) 
{
	digits = N%10;
	// обработка digit
	N = N/10;
}
```

$$\sum^{N}_{i=1}i$$
```c++
int N = 20, i=1;
long S = 0;
while (i<=N)
{
	S = S+1;
	i++;
}
```

### Цикл с постусловием
#### Синтаксис
```c++
do
	тело_цикла;
while (условие);
```
#### Семантика
выполняется тело цикла, затем вычисляется условие и если оно не равно 0, то снова выполняется тело цикла. Если результат условия равен нулю, от выполняется следующий оператор.
Основное **отличие** от цикла с предусловием - тело цикла выполнится **хотя бы один** раз.
#### Примеры
```c++
int n, f=1, i=0;
cin >> n;
do 
{
	i++;
	f *= i;
} while (i <= n);
cout << f;
```
