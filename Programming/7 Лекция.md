В квадратной матрице существуют понятия главной и побочной диагоналей

| i == j | На главной   | i = N-j-1 | На побочной   |
| ------ | ------------ | --------- | ------------- |
| i < j  | Выше главной | i < N-j-1 | Ниже побочной |
| i > j  | Ниже главной | i > N-j-1 | Выше побочной |

*Главная диагональ*

| ------- |             |             |
| ------- | ----------- | ----------- |
|         | **-------** |             |
|         |             | **-------** |

# Сортировки 
## Общая постановка задачи
Пусть на некотором множестве $X$ задано отношение меньше либо равно, обладающее свойствами:
1. $\forall a,b \in X(a<=b \cup (b<=a)$
2. $\forall a,b \in X (\dots)$
Пусть $a_1, .., a_n \in X$ $i_1, i_2, ...,i_n$ - такая перестановка натуральных чисел 1,2, ... n, что последовательность $a_i, ..., a_n \in X$ такова, что $a_{i_1} <= a_{i_2} <= ... <= a_{i_n}$ Тогда последовательность 2 называется упорядоченной или отсортированной
Методы приводящие от посл. 1 к посл 2, называются методами сортировки

Для реализации сортировки в элементе последовательности необходимо выделить или искусственно добавить **поле**, имеющее уникальное значение, такое поле называется **ключом сортировки**

### Существуют 2 типа сортировок
1. Если сортируемый массив полностью помещается в оперативную память, то такая сортировка называется **внутренней**.
2. В противном случае - это **внешняя** сортировка
