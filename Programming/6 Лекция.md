## Обработка одномерных массивов
Заключается в их переборе, который характеризуется:
- Направлением перебора;
- Количеством одновременно обрабатываемых элементов;
- Характером изменения индексов. 

### По направлению перебора
Массивы обрабатываются:
1) слева направо (от начала массива к его концу);
2) справа налево (от конца массива к началу);
3) от обоих концов к середине.
### Индексы
- могут меняться линейно (с постоянным шагом);
- нелинейно (с переменным шагом).

### Схемы организации перебора
1) `for (int i = 0; i < n; i++) {обработка a[i]}`
2) `for (int i = 0; i < n; i+=step) {обработка a[i]}`
3) `for (int i = 0; i < n; i--) {обработка a[i]}`

### Пример 1
```c++
#define N 100
int main() {
	int a[N];
	for (int i = 0; i < N; i++) {cin >> a[i];}
	int s=0;
	for (int i = 0; i<N; i++) {if (a[i]%2!=0) s += a[i];}
	cout << s;
	return 0;
}
```
### Пример 2
```c++
#define N 100
int main() {
	int a[N];
	for (int i = 0; i < N; i++) {cin >> a[i];}
	int s=0;
	for (int i = 0; i<N; i+=2) {if (a[i] > 0) s += a[i];}
	cout << s;
	return 0;
}
```
### Схемы парной обработки элементов массива
1) 
```c++
int i = 0; J=N-1
while (I<J) {
	// обработка a[i] и a[J]
	i++; J--;
}
```
2)  Элементы массива можно обрабатывать по два элемента, двигаясь от начала к концу с шагом 1 (т.е. обрабатываются пары элементов `a[0]` и `a[1]`, `a[1]` и `a[2]` и т.д.):
```c++
for (I = 0; I < N-1; I++) {
	// обработка a[I] и a[I+1]
}
```
Пример
```c++
int k = 1;
for (I = 0; I<N-1; I++) {
	if (a[I] >= a[I+1]) k = 0;
}
if (k == 0) cout << "Не упорядочен" << endl;
else cout << cout << "Упорядочен" << endl;
```
3) Элементы массива можно обрабатывать по два элемента, двигаясь от начала к концу с шагом 2 (т.е. обрабатываются пары элементов `a[0]` и `a[1]`, `a[2]` и `a[3`] )
Пример
```c++
int i = 0; k = 0
while (i < N-1) {
	if (a[i] * a[i+1] < 0) k++;
	i += 2;
}
```

### Задачи 2-го класса
задачи, которые меняют порядок следования элементов.
```c++
int R = a[I];
a[I] = a[J];
a[J] = R;
// обмен a[I] и a[J] элементов массива.
```

Пример 1: обратить массив
```c++
// ввод массива a
for (int i = 0; j = n-1; i < j; i+=, j--) {
	int r = a[i];
	a[i] = a[j];
	a[j] = r;
}
// вывод массива a
```
Пример 2: поменять элементы в парах, если нарушается условие возрастания в паре.
```c++
int i = 1;
while (i < N-1) {
	if (a[i] > a[i+1]) {
		c = a[i];
		a[i] = a[i+1];
		a[i+1] = c;
	}
	i += 2;
}
```
