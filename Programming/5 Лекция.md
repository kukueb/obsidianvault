## Цикл с параметром
### Семантика
Инициализатор выполняется **один** раз в начале выполнения цикла, затем проверяется условие и при его ненулевом значении выполняется тело цикла. Затем выполняется итерация, изменяющая параметры цикла. 

### Примеры
1) Уменьшение параметра:
```c++
for (n = 10; n > 0; n--) {
	оператор;
}
```
2) Изменение шага корректировки:
```c++
for (n = 2; n < 60; n += 13) {
	оператор;
}
```
3) Возможность проверять условие отличное от условия, которое налагается на число итераций
```c++
for (num=1; num*num*num<216; num++){
	оператор;
}
```
4) Коррекция может осуществляться не только с помощью сложения или вычитания:
```c++
for (d=100.0; d<150.0; d*=1.1) {
	<тело цикла>;
}

for (x=1; y<=75; y=5*(x++)+10) {
	оператор;
}
```

## Операторы перехода
1) `break` - оператор прерывания цикла. Оператор **досрочного выхода** из цикла
```c++
{
	<операторы>;
	if (<выражение_условие>) break;
	<операторы>;
}
```
### Пример
```c++
for (s=0, i=1; i<100; i++) {
	cin >> x;
	if (x == 0) break; // если ввели 0, то суммирование заканчивается
	s += x;
}
```
---
2) `continue` - переход к следующей итерации цикла. Чаще всего используется при **ветвлениях в цикле**
```c++
for (k=0,s=0,x=1; x != 0) {
	cin >> x;
	if (x<= 0) continue;
	k++; s+=x;
}
```
---
3) Оператор `goto`
Оператор `goto` имеет формат:
`goto` *<метка>*
Осуществляется переход на строку, помеченную меткой, как вниз, так и вверх по тексту программы
```c++
int k;
goto m;
...
{
	int a=3, b=4;
	k = a+b;
	m: int c = k+1;
	...
}
```
### Свойства оператора goto

`return [выражение];`

## Обработка последовательностей. Программирование арифметических циклов

```c++
cin >> N;
for (i=0; i < N; i++) {
	cin >> x;
	обработка x;
}
```

#### Пример 1.
*Дана последовательность из N целых чисел. Найти сумму четных чисел последовательности.*
```c++
cin >> N;
int s=0;
for (i=0; i<N; i++) {
	cin >> x;
	if (x%2 == 0) s+=x;
}
cout << s;
```
---
#### Пример 2.
*Дана последовательность из N целых чисел. Найти произведение нечетных чисел последовательности, за которыми идёт отрицательное число.*
```c++
cin >> N;
int p = 1;
cin >> a;
for (i=1; i<N; i++) {
	cin >> b;
	if (a%2 != 0 && b < 0) p *=a;
	a = b;
}
```

### Итерационные циклы
```c++
cin >> x;
while (x != маркер) {
	обработка x;
	cin >> x;
}
```

#### Пример
*Дана последовательность целых, оканчивающаяся нулем.
Найти минимальный элемент*
```c++
void main() {
	int a,min;
	cout << "\nEnter a";
	cin >> a;
	min=a;
	while (a!=0) // for (;a!=0;) // while (a)
	{
		cout << "\nEnter a";
		cin >> a; // поменять строки местами!
		if (a!=0 && a<min) min = a; 
	}
	cout << "\n min=" << min << "\n";

```

*Найти кол-во пар чисел последовательности разных знаков.*
```c++
int k=0;
cin >> x;
while (x!=0) {
	cin >> y;
	if (x*y < 0) k++;
	x = y;
}
```

## Регулярные типы. Одномерные массивы
Массив - это набор однотипных проиндексированных элементов. Нумерация начинается с 0
*<тип данных> <имя массива>\[число элементов\]*

```c++
int array_int[100]; //одномерный массив 100 целочисленных элементов
double array_d[25]; 
```

Обращение к отдельному элементу массива осуществляется по номеру его индекса
```c++
a[55] - индекс задаётся как константа,
a[i] - индекс задаётся как переменная,
a[2*i] - индекс задаётся как выражение.
```

```c++
int N=100;
float array_f[N]; // ошибка, так нельзя
```

#define N 100
int main() {
	float array_f[N]; // РАБОТАЕТ, ВААА
