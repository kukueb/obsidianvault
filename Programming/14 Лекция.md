## Указатели на функции
Указатель на ф-цию хранит адрес функции, то есть адрес первого байта памяти, то есть на первый байт памяти, где располагается код функции.

`тип (*имя_указателя)(типы_параметров);`

```cpp
void hello();
void goodbye();
int main() {
 void (*message)(); // определение указателя на функцию
 
 message = hello;
 message();
 message = goodbye;
 message();
}
```

Указатель может работать только с функциями у которых такой же тип результата и типы параметров
```c++
void (*message)() { hello }; // указывает на функцию hello
void (*message)() = hello;

auto message { hello };
auto message2 = hello;
```

```c++
int sum (int, int);
int subtract (int, int);

int main() {
	int a {10};
	int b {5};
	int (*operation)(int, int) {sum};
	int result = operation(a,b);
	// resilt = (*operation)(a,b); //альтернативный вариант
	cout << "result= " << result << endl; // result = 15
}
```

## Динамические объекты
В C++ можно использовать различные объекты, которые отличаются по **типу использования памяти**.
Так глобальные переменные создаются при запуске программы и уничтожаются по её окончанию.
**Локальные** объекты автоматически **создаются в начале блока**, где они описаны и **уничтожаются в конце блока**.

Существуют **статические** и **динамические** объекты
**Статические объекты** размещаются в **стеке** и существуют до окончания блока

**Динамические объекты** можно создавать и удалять явным образом в любом месте программы.
Такие объекты размещаются в динамической памяти.

#### Преимущества динамической памяти
Под объект выделяется столько памяти, сколько необходимо.

При **статическом** подходе часто необходим *неделимый* фрагмент памяти.

Для управления динамической памятью используют операторы 
```c++
new
// и 
delete
```

**`new`** - Выделяет место в динамической памяти и возвращает указатель на выделенный блок
**`delete`** - получает указатель на динамический объект и удаляет его из памяти.

Создание динамического объекта:
```c++
int *ptr {new int};
// или
int *ptr = new int;
```

```c++
int *ptr {new int{5}};
//альтернативные варианты

// int *ptr = new int{5};
// int *ptr {new int(5)};
// int *ptr = new int(5);
cout << *ptr << endl; //5
```

### Освобождение памяти
```c++
int *ptr {new int{5}}; //выделяем память
cout << "*ptr = " << *ptr << endl; //*ptr = 5
delete ptr;
```

```c++
int* createPtr(int value) {
	int *ptr {new int{value}};
	return ptr;
}

void usePtr() {
	int *obj = createPtr(10);
	cout << *obj << endl; // 10
	delete obj;
}
void main() {
	usePtr();
}
```

### Многомерные динамические массивы

Пример 1. Удалить из матрицы строку с номером K
```c++
void main() {
	int n, m;
	int i, j;
	cin >> n; // строки
	cin >> m; // столбцы
	
	// выделение памяти
	int **matr = new int* [n]; // массив указателей на строки
	for (i = 0; i < n; i++) for (j = 0; j < m; j++) // память под матрицы
		matr[i][j] = rand()%10; // заполнение матрицы

	// удаление
	for (i = 0; i < n; i++)
		delete matr[i];
	delete [] matr;
}
```
