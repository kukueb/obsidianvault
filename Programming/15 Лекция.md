## Классы в C++

### Понятие класса
**Класс** это *абстрактный* тип данных, состоящий из полей, в которых хранятся данные класса, и методов, которые представляют собой действия над полями. Объединение полей и методов в одну структуру данных называется **Инкапсуляцией**. 

Вторым базовым свойством **ООП** (Объектно Ориентированное Программирование) является **Наследование**. То есть классы могут образовывать иерархию, в которой свойства родителя автоматически передаются классу наследнику.

Потомки класса могут переопределять поля и методы, такой процесс называется **Полиморфизмом**.

*ООП:
1) Инкапсуляция 
2) Наследование
3) Полиморфизм

Экземпляр класса называется **Объект**

---
```c++
class имя_класса {
	// компоненты
};
```

```c++
class Person {
public:
	string name;
	unsigned age;
	void print() {
		cout << "Name: " << name << "\tAge: " << age << endl;
	}
};

int main() {
	Person person;
	person.name = "Tom"; // объект.компонент - обращение к экземпляру класса
	person.age = 38;
	person.print();
}
```

### Модификаторы доступа
Если компоненты класса необходимо сделать доступными из вне, они размещаются в секции **`public`**. Закрытые компоненты размещаются в секции **`private`**.

Если в классе секции не указаны, то все компоненты считаются **закрытыми**.

### Указатели на объекты классов
На объекты классов можно определять указатели, но для обращения к компонентам такого экземпляра необходимо использовать *стрелку*

```c++
class Person {
  public:
    string name;
    unsigned age;
    void print() {
      cout << "Name: " << name << "\tAge: " << age << endl;
    }
};

int main() {
  Person person;
  Person* ptr = &person;
  // обращаемся к полям и функциям объекта через указатель
  ptr->name = "Tom";
  ptr->age = 22;
  ptr->print();
  // обращаемся к поляи объекта
  cout << "Name: " << person.name << "\tAge: " << person.age << endl;
}
```


### Конструкторы
Конструктор - это специальная функция, имя которой совпадает с именем класса и которая не возвращает никакого значения, но может инициализовать поля класса.

При каждом создании нового объекта класса конструктор **вызывается автоматически**.
```c++
class Person {
  public:
    string name;
    unsigned age;
    void print() {
      cout << "Name: " << name << "\tAge: " << age << endl;
    }
};

int main() {
  Person person; // Вызов конструктора
}
```

```c++
class Person {
  public:
    string name;
    unsigned age;
	
    void print() {
      cout << "Name: " << name << "\tAge: " << age << endl;
    }
	
	Person(string p_name, unsigned p_age) {
		name = p_name;
		age = p_age;
		cout << "Person has been created" << endl;
	}
};

int main() {
	Person person("Tom", 38); // Вызов конструктора
	person.print();
	
	Person tom{"Tom", 38};
	Person bob{"Bob", 42};
	Person sam{"Sam", 25};
	
	tom.print();
	bob.print();
	sam.print();
	
}
```

### Определение нескольких конструкторов
```c++
class Person {
  string name{};
  unsigned age{};

public:
  void print() {
    cout << "Name: " << name << "Age: " << age << endl;;
  }

  Person(string p_name, unsigned p_age) {
    name = p_name;
    age = p_age;
  }

  Person(string p_name) {
    name = p_name;
    age = 18;
  }

  Person() {
    name = "Undefined";
    age = 18;
  }
};

int main() {
	Person tom{"Tom", 38};
	Person bob{"Bob"};
	Person sam;
	
	tom.print(); // Person(string p_name, unsigned p_age)
	bob.print(); // Person(string p_name)
	sam.print(); // Person()
}
```

```c++
class Person {
  string name{};
  unsigned age{};
public:
  void print() {

  }

  Person(std::string p_name, unsigned p_age) {

    age = p_age;
  }

// Вызывается первый конструктор
  Person(std::string p_name): Person(p_name, 18) { 
    cout << "Second" << endl;
  }
  
// Вызывается второй конструктор
  Person(): Person("Undefined"){ 
    cout << "Third" << endl;
  }
  
};

int main() {
	Person sam;
	sam.print();
}
```

### Объявление и определение функций класса
`имя_класса::имя_функции(параметры){тело_функции}`

```c++
class Person {
  string name{};
  unsigned age{};
public:
  void print();
  Person(std::string p_name, unsigned p_age);
};

// конструктор
Person::Person(string p_name, unsigned p_age) {
  name = p_name;
  age = p_age;
}

void Person::print() {
  cout << "Name: " << name << "\tAge: " << age << endl;
}

int main() {
	Person tom{"Tom", 38};
	tom.print(); // Name: Tom Age: 22
}
```

### Деструктор
Деструктор класса вызывается при **уничтожении** объекта. Его имя совпадает с именем класса, но ставится тильда.

Деструктор *не имеет параметров*, освобождает память, выделенную под объекты.
Вызывается автоматически при удалении объекта.
```c++
~имя_класса() {
	// код деструктора
}
```

```c++
class Person {
  public:
    Person(string p_name) {
      name = p_name;
    }

    ~Person() {
      cout <<"Person " << name << " destroyed" << endl;;
    }
};

int main() {
  {
    Person tom {"Tom"};
    Person bob {"Bob"};
  } // объекты Tom и Bob уничтожаются

  Person sam {"Sam"};
} // объект Sam уничтожается
```
