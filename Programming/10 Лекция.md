## Передача параметров по значению
При обработке механизма формальных - фактических параметров
передавать собственно параметр можно по значению, по адресу и по ссылке
При передаче параметров по значению, значение фактического параметра размещается в стеке и формальный параметр обращается к этому значению.
Даже если в функции планируется изменение формального параметра, то это новое значение вне функции недоступно.

## Передача параметров по адресу
Такой способ используется в языке C, в этом случае в качестве фактического параметра указывается адрес фактического параметра
```c++
void swap (int v1, int v2) {
	int tmp = v2;
	v2 = v1;
	v1 = tmp;
}
// swap() не меняет значений своих аргументов! 
```

```c++
//pswap() обменивает значния объектов
//адресуемых указателями v1 и v2
void pswap(int *v1, int *v2) {
	int tmp = *v2;
	*v2 = *v1;
	*v1 = tmp;
}

int main() {
	int i = 5, j = 10;
	pswap(&i, &j); // i = 10, j = 5
	return 0;
}

// & - взять адрес
// * - взять значение по адресу
```

## Передача параметров по ссылке
```c++
void rswap(int &v1, int &v2) {
	int tmp = v2;
	v2 = v1;
	v1 = tmp;
}

int main() {
	int i = 10, j = 20;
	rswap(i, j); // i = 20, j = 10
	return 0;
}
```

# Константная ссылка
Если необходимо запретить изменение параметра внутри функции, используется модификатор const:
```c++
void decr(const int& n) {
	cout << n; //читать можно
	n--; // изменять нельзя, будет ошибка CE
}
```

# Автоматическое выведение типа параметров
```c++
void sum (auto, auto);

int main() {
	const int n1(3);
	const int n2(4);
	sum(n1, n2); // 3 + 4 = 7
	const double d1(3.3);
	const double d2 (4.4);
	sum(d1,d2); // 3.3 + 4.4 = 7.7
	return 0;
}
void sum(auto a, suto b) {
	auto result = a+b;
	cout << a << " + " << b << " = " << result << endl;
}
```

При использовании спецификатора `auto` в стандарте C++20 тип формальных параметров подбирается в зависимости от типа фактических параметров.

## Преобразования типов
при передаче параметров можно автоматически преобразовывать один тип к другому с потерей точности, но при передаче параметров по ссылке, неявные преобразования исключены.
```c++
void printVal(int);
void printRef(int&);

int main()
{
	double value(3.1415);
	printVal(value); // 3
	printRef(value); // Ошибка!
}

void printVal(int n) {
	cout << n << endl;
}
void printRef(int& n) {
	cout << n << endl;
}
```

# Локальные и глобальные переменные
Все величины, описанные внутри функции, а так же её параметры, являются локальными.
Область их действия - функция.
Память под локальные переменные выделяется только при вызове функции.

Глобальные переменные - это переменные описанные вне функции. 
Они видны во всех функциях, где нет локальных переменных с теми же именами.
```c++
int a,b; //глобальные переменные

void change() {
	int r; // локальная переменная
	r = a; a = b; b = r;
}

void main() {
	cin >> a >> b
	change();
	cout << "a = " << a << "b = " << b << endl;
}
```

_Пример._ Работа с глобальными и локальными переменными.
```c++
int var = 5;
int main() {
	int var = 10; // Переопредение переменной в локальную
	cout << var << ::var << endl; // Результат: 10 5
	//             ^^^^^ глобальная
	return 0;

```

При переопределении глобальных переменных одноимёнными локальными, работа осуществляется с локальной переменной, однако к глобальной переменной можно получить доступ, указав `::` 

_Пример._ Использование статических переменных.
```c++
void iter(void);

int main() {
	for (int i = 0; i < 4; ++i) {
		iter();
	}
	return 0;
}

void iter(void) {
	int var = 1; // всё время выделяется новая память
	static int var_st = 1; // один раз выделяется память
	cout << var++ << var_st++;
}

// В результате
// var = 1, var_st = 1
// var = 1, var_st = 2
// var = 1, var_st = 3
// var = 1, var_st = 4
```

Статические переменные определённые и имеющие область видимости внутри фигурных скобок `{ }`  не исчезают за их пределами и сохраняют значения.

# Параметры со значениями по умолчанию
Некоторые параметры можно объявить как значение по умолчанию и при вызове функции не указываете значение. Перечень таких параметров всегда находится в конце списка параметров

```c++
// int pow(const int& a, const int& p = 2) {
//                                  ^^^^^
int pow(const int& a, const int& p) {
	int res = 1;
	for (int i = 1; i <= p; i++) {
		res = res*a;	
	}
	return res;
}
```
