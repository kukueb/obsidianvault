### Сортировка Хоара (Быстрая сортировка)
Время работы $O(n\log n)$.
**Идея**: в сортируемой последовательности необходимо найти такой элемент с номером $f$, что все элементы, расположенные левее его, меньше по значению, а все элементы, расположенные правее, больше по значению.
Такой элемент называется опорным, и он уже находится на своём месте в отсортированной последовательности.
Такой элемент можно получить за несколько перестановок. Далее алгоритм применяется рекурсивно к левой и правой частям массива.
В качестве начальной позиции $f$ можно выбирать средний элемент или произвольный.

```c++
int a[10], n;
void hoarasort(int*, int, int);

int main() {
	int i;
	cin >> n;
	for (i = 0; i < n; i++)
		cin >> a[i]
	
	hoarasort(a, 0, n-1);
	
	for (i = 0; i < n; i++)
		cout << a[i]
	return 0;
}

void hoarasort(int* a, int first, int last) {
	int i = first, j = last;
	int tmp, f = a[(first + last) / 2];
	do {
		while (a[i] < f) i++;
		while (a[j] > f) j--;
		if (i <= j) {
			if (i < j) {
				tmp = a[i];
				a[i] = a[j];
				a[j] = tmp;
			}
			i++;
			j--;
		}
	} while (i <= j);
	if (i < last) hoarasort(a, i, last);
	if (first < j) hoarasort(a, first, j);
}
```

## Введение в ИИ

**Машинное обучение (Machine Learning, ML)** - это область искусственного интеллекта, в которой компьютеры учатся решать задачи на основе данных **без явного программирования**.
*Ключевые идеи:*
*Вместо жёстких правил (как в традиционном программировании) ML - модели обучаются на примерах.*
*Обучение происходит через оптимизацию (поиск наилучших параметров модели)*
*Основные типы задач:*
 - *Классификация (например, распознавание цифр).*
 - *Регрессия (предсказание чисел, например, цены акций).*
 - *Кластеризация (группировка данных без учителя).*

| Критерий   | Обучение с учителем                         | Обучение без учителя                  |
| ---------- | ------------------------------------------- | ------------------------------------- |
| **Данные** | Ксть размеченные данные ($X \rightarrow Y$) | Нет меток, только $X$                 |
| **Задачи** | Классификация, регрессия                    | Кластеризация, уменьшение размерности |
| **Пример** | Распознавание цифр                          | Группировка клиентов по покупкам      |
### Градиентный спуск
Математический метод оптимизации для минимизации ошибки. Ключевой алгоритм для обучения нейронных сетей и машинного обучения. Это метод оптимизации, используемый для минимизации функции потерь (loss function) в машинном обучении.

**Функция потерь** - мера ошибки модели (Например, $f(x) = x^2$)

**Градиент ($\nabla f(x)$)** - вектор частных производных, указывающий направление наибольшего роста функции

**Шаг обучения (learning_rate)** - насколько сильно мы корректируем параметры на каждой итерации.

Процесс:
1) Нейросеть делает прогноз.
2) Сравнивает с правильным ответом.
3) Вычисляется **ошибка**.
4) Градиентный спуск показывает, как изменить веса модели, чтобы ошибка уменьшилась.

Формула обновления:
$$
x_{new} = x_{old} - \eta \nabla f(x_{old})
$$
где $\eta$ - шаг обучения

**Задача:**
Дана квадратичная функция $f(x) = x^2 - 4x + 5$. Найти её минимум, используя метод градиентного спуска

```c++
#include <iostream>
#include <cmath>
using namespace std;

double f(double x) {
	return x*x - 4*x + 5;
}

double df(double x) {
	return 2*x - 4;
}

int main() {
	double x = 10.0;
	double learning_rate = 0.1
	double epsilon = 1e-6;
	int max_iterations = 1000;
	
	// Градиетный спуск
	int iteration = 0;
	double prev_x, gradient;
	
	do {
		prev_x = x;
		gradient = df(x);
		x = x - learning_rate * gradient;
		
		iteration++;
		
		//Вывод информации о текущей итерации
		cout << "Итерация " << iteration << ": x = " << x;
		cout << ", f(x) = " << f(x) << ", градиент = " << gradient << endl;
	} while (f(abs(x - prev_x)) > epsilon && iteration < max_iterations);
	
	return 0;
}
```

### Указатели
Когда компилятор обрабатывает определение переменной (`int i = 10`), он выделяет память в соответствии с типом, инициализирует эту память значением. Далее в программе все обращения к имени переменной заменяются адресом выделенной области.
Переменные, хранящие адреса областей памяти, называются **Указателями**.
В C++ различают:
- Указатель на **функцию**
- Указатель на **объект**
- Указатель на `void`
Указатель не является самостоятельным типом, однако связан с типом хранимого значения.

#### Указатель на функцию
Содержит адрес в сегменте кода, по которому располагаются команды функции. На этот адрес передаётся управление при вызове функции
`тип (*имя) (список формальных параметров)`
Указатель на функцию имеет тип возвращаемого значения функции.
Например, объявление:
`int (*fun) (double, double)`
