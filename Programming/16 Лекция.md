## Шаблоны функций
Шаблоны функций позволяют определить конструкции, которые используют определённые типы, не определённые в момент написания кода.

Это универсальные конструкции, не зависящие от конкретных типов.
 
Пример, определим функцию для сложения двух чисел int, double и string
```c++
int add(int, int);
double add(double, double);
string add(string, string);

int main (int argc, char *argv[]) {
  cout << "int: " << add(4,5) << endl;
  cout << "double: " << add(4.4,5.5) << endl;
  cout << "string: " << add("hel","lo") << endl;

  return 0;
}


int add(int x, int y){
  return x+y;
}
double add(double x, double y) {
	return x + y
}
string add(string x, string y) {
	return x + y
}
```

```c++
template<typename T> T add(T, T);

int main (int argc, char *argv[]) {
  cout << "int: " << add(4,5) << endl;
  cout << "double: " << add(4.4,5.5) << endl;
  cout << "string: " << add("hel","lo") << endl;

  return 0;
}

template<typename T> T add(T a, T b) {
  return a + b;
}
```

Можно использовать ссылки, указатели и массивы в качестве параметров шаблона.
```c++
template<typename T> T add(const T&,const T&);

int main (int argc, char *argv[]) {
  cout << "int: " << add(4,5) << endl;
  cout << "double: " << add(4.4,5.5) << endl;
  cout << "string: " << add("hel","lo") << endl;

  return 0;
}

template<typename T> T add(const T& a,const T& b) {
  return a + b;
}
```

## Структуры
Структура - это произвольный тип данных, который может объединять переменные различных типов.

```c++
struct имя_структуры {
	компоненты_структуры;
};
```

```c++
struct person {
	unsigned age;
	string name;
};

int main() {
	person tom;
	tom.name = "Tom";
	tom.age = 34;
	cout << "Name: " << tom.name << "\tAge: " << tom.age << endl;
}
```

### Обращение к элементам структуры
`имя_переменной_структуры.имя_элемента`
`person tom { 34, "Tom" }; // инициализвация структуры`

Структуры могут быть вложенными
```c++
struct tag_fio {
  char last[100];
  char first[100];
  char otch[100];
};

struct tag_people {
  struct tag_fio fio; // tag_fio fio;
  char job[100];
  int old;
};

int main (int argc, char *argv[]) {
  struct tag_people man = { // tag_people man = {
    {"Иванов", "Иван", "Иванович"},
    "Электрик",
    50
  };

  return 0;
}
```

```c++
void show_struct(struct tag_people man) {
  cout <<"Имя: " << man.name << endl;
  cout <<"Профессия: " << man.job << endl;
  cout <<"Возраст: " << man.age << endl;
}
```

Можно делать массивы структур
```c++
void show_struct(struct tag_people men[], int size) {
  // реализация
}
```

## Строки (на экзамене привести 5 любых функций)
```c++
#include <string>
```

```c++
string s1; // переменная с именем s1 типа string
string s2 = "This is a string variable"; // объявление с инициализацией
```
Максимальная длина строки (теоретически) - 256 символов

```c++
s1 = s2; // s1 = This is a string variable" s2 = "New text";
```
Длину строки возвращает функция класса size()
```c++
cout << "Длина " << s2.size() << endl;
string s1; // пустая строка
```

проверить пустоту строки можно сравнив длину с 0
`if (!s1.size())`
или
`if (s1.empty())`

```c++
s1 = "hel";
s2 = "lo";
s1 += s2 // hello
```

С объектами класса string можно использовать нижеследующие операторы

`=` - присваивание
`+` - конкатенация
`+=` - присваивание с конкатенацией
`==` - равенство
`<` - меньше
`<=` - меньше или равно
`>` - больше
`>=` - больше или равно
`[]` - индексация
Ввод строки до пробела:
`cin >> s;`
Ввод строки целиком
`getline(cin, s);`

С помощью методов length() и size() можно узнать размер строки, то есть из скольких символов она состоит.
Если строка пустая, то она содержит 0 символов. В этом случае можно применить метод empty() - он возвращает true, если строка пустая

### Преобразование в строку
```c++
unsigned age {38};
string message {"Age: " + to_string(age)};
cout << message << endl; // Age: 38
```

Функции преобразования строки в число:
 - `stoi()`  -> int
 - `stol()` -> long
 - `stoll()` -> long long
 - `stoul()` -> unsigned long
 - `stoull()` -> unsigned long long
 - `stof()` -> float
 - `stod()` ->  double
 - `stold()`  -> long double
