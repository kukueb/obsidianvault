# Контейнерный класс `Vector`
## Методы работы с `Vector`
- `empty()` - `true`, если пустой, `false` иначе
- `size()` - кол-во элементов вектора
- `max_size()` - максимально возможное кол-во элементов вектора
- `reserve()` - устанавливает минимально возможное кол-во элементов вектора
- `capacity()` - кол-во элементов вектора, которое он может содержать до того, как ему потребуется выделить больше памяти
- `clear()` - удаляет все элементы, при этом размерность вектора остаётся без изменений

```cpp
void print_vector(vector<int> b) {
  if (b.empty()) return;
  for (int i = 0; i < b.size(); ++i) {
    cout << b[i] << " ";
  }
}
```
Ёмкость контейнера - это максимальное количество элементов, которое может вместить контейнер без дополнительного выделения памяти. 
```cpp
vector<int> a;
for (int i = 0; i < 24; ++i) {
	a.push_back(i);
}
cout << "a: размер: " << a.size() << " ёмкость: " << a.capacity() << endl; 
// размер 24, ёмкость 32
```

### `insert()`
Функция `insert()` - вставка элемента в начало контейнера
1 форма
	1 параметр - итератор, который показывает на позицию в контейнере
	2 параметр - что мы хотим добавить перед позицией из 1 параметра
```cpp
  vector<int> a;
  for (int i = 0; i < 4; ++i) {
    a.push_back(i);
  }
  a.insert(a.begin(), 5); // 5, 0, 1, 2, 3
```
2 форма
	1 параметр - итератор
	2 - сколько раз вставить
	3 - само значение
```cpp
  a.insert(a.end(), 3, 77) // 5, 0, 1, 2, 3, 77, 77, 77
```
3 форма
	добавление нескольких элементов
	1 - итератор
	2 - начало вектора
	3 - до какого
```cpp
int b[2] = {9, 99};
a.insert(a.begin(), b, b+2); // 9, 99, 5, 0, 1, 2, 3, 77, 77, 77
```

### `erase()`
Функция удаление элементов из массива - `erase()`
1 форма
	удалить единственный элемент по итератору
```cpp
a.erase(a.begin()+2); // 9 99 0 1 ...
```
2 форма
	1 - начало удаления
	2 - конец удаления
```cpp
a.erase(a.begin(), a.begin()+2) // 5 0 1 ...
```

### `pop_back()`
Удалить последний элемент - `pop_back()`

### Метод `swap()`
```cpp
svec1.swap(svec2);
```
Размеры при этом становятся одинаковыми в пользу большего размера

# Контейнерный класс `list`
Это двунаправленный список, в контейнере не определена операция доступа по индексу и для того, чтобы обратиться к элементу внутри списка, необходимо выполнить перебор
```cpp
#include <list>
list<int> list1; // пустой список
list<int> list2(5); // список list2 состоит из 5 чисел, каждый элемент имеет значение по умолчанию
list<int> list3(5,2); // список list3 состоит из 5 чисел, каждое из которых равно 2
```

**Методы работы со списком**

*   `front()`
*   `back()`
*   `size()`
*   `empty()`
*   `resize(n)`
*   `resize(n, value)`
*   `assign(il)`
*   `assign(n, value)`
*   `assign(first, last)`
*   `push_back(value)`
*   `push_front(value)`
*   `insert(pos, value)`
*   `insert(pos, n, value)`
*   `insert(pos, first, last)`
*   `insert(pos, values)`
*   `clear()`
*   `pop_front()`
*   `pop_back()`
*   `erase(pos)`
*   `erase(first, last)`

### `resize(n)`
Оставляет в списке первые `n` элементов

### `assign()`
1 форма - `assign(il)`:
- заменяет содержимое контейнера элементами из списка инициализации `il`
2 форма - `assign(n, value)`:
- Заменяет содержимое контейнера `n` элементами, которые имеют значение `value`
3 форма - `assign(begin, end)`:
- Заменяет содержимое контейнера элементами из диапазона на начало и конец которого указывают итераторы `begin` и `end`

### Добавления
`push_back(val)`
`push_front(val)`

### ещё функции
`clear(p)` - удаляет все элементы
`pop_back()`
`pop_front()`
`erase(p)` - удаляет элемент, на который указывает итератор `p`. Возвращает итератор на элемент, следующий после удалённого, или на конец контейнера, если удалён последний элемент
`erase(begin, end)` - удаляет элементы из диапазона
