# Множества (`set`)
## Неупорядоченное множество `unordered_set`
поддерживает практически все те же функции, только не упорядочивает элементы

```cpp
#include <unordered_set>

int main() {
	unordered_set<int> numbers {3,2,5,4};
	numbers.insert(1);
	numbers.insert(6);
	
	for (int n : numbers) {
		cout << n << "\t"
	}
}
```

### Пример
```cpp
#include <iostream>
#include <set>

using namespace std;

int main() {
  set<int> positive;
  set<int> negative;

  int x;
  for (cin >> x; x != 0; cin >> x) {
    if (x > 0)
      positive.insert(x);
    else
      negative.insert(x);
  }

  cout << "Ответ" << endl;
  for (set<int>::iterator it = positive.begin(); it != positive.end(); ++it) {
    cout << *it << " ";
  }
  cout << endl;
  for (set<int>::reverse_iterator it = negative.rbegin(); it != negative.rend(); ++it) {
    cout << *it << " ";
  }
  cout << endl;

  return 0;
}
```

# Контейнер `map`
Контейнер `map` каждое значение связывает с определённым ключом. Ключи могут иметь только уникальные значения. Доступ к элементу осуществляется по ключу.

```cpp
#include <map>
map <<тип_ключа>, <тип_значения>> <имя>;
map <string, int> ar;
```

Для обращения к элементам словаря - получение или изменения их значений, так же, как в массиве или векторе, применяется оператора индексирования `[]`

### Пример
```cpp
#include <iostream>
#include <map>

using namespace std;

int main() {
  map<string, unsigned> products;
  // установка значений
  products["bread"] = 30;
  products["milk"] = 80;
  products["apple"] = 60;

  // получение значений
  cout << "bread\t" << products["bread"] << endl;
  cout << "milk\t" << products["bread"] << endl;
  cout << "apple\t" << products["apple"] << endl;
  return 0;
}
```

## Перебор элементов
Для работы со словарём удобно использовать специальный объект `pair<const Key, Value>` , который хранит ключ и значение.
*`pair<const std::string, unsigned int>`*
С помощью полей `first` и `second` данного объекта получают соответственно ключ и значение элемента:
```cpp
for (const auto &element : products)
	cout << element.first << "\t" << element.second << endl;
```

Со стандарта C++17:
```cpp
for (const auto &[product, price] : produts)
	cout << product << "\t" << price << endl;
```

### Пример
Найти наиболее часто встречающееся число
```cpp
#include <iostream>
#include <map>

using namespace std;

int main() {
  int n;
  cin >> n;
  map<long, long> m;
  for (long i = 0; i < n; ++i) {
    long x;
    cin >> x;
    ++m[x];
  }
  if (n == 0) {
    cout << "Не найдено" << endl;
    return 0;
  }
  pair<long, long> max = make_pair(m.begin()->first, m.begin()->second);

  for (map<long, long>::iterator it = m.begin(); it != m.end(); ++it) {
    if (max.second < it->second) {
      max = make_pair(it->first, it->second);
    }
    cout << "Значение " << max.first << " встречается " << max.second
         << " раз(а)" << endl;
  }
  return 0;
}
```

С переменной типа `pair` можно работать как с единым целым
```cpp
pair<int,int> a, b;
...
b = a;
```
Но при вводе-выводе переменных, типа `pair` нельзя выводить или вводить значение переменной одной командой, то есть **нельзя** сразу же вывести два значения типа:
```cpp
cin >> a; cout << a; // !!!
```

Пары сравниваются в лексикографическом порядке. Сначала выполняется сравнение по полю `first`. Если эти поля равны, то выполняется сравнение по полю `second`

С парой, как с целым (то есть одновременно с двумя полями структуры) можно выполнять следующие операции: `=, ==, !=, <m, <=, >, >=`, а также использовать в качестве аргумента функции `swap`

```cpp
vector< pair<int,int> > a // пробел между > >
```

### Удаление `erase()`
элемент удаляется по ключу
```cpp
products.erase("milk");
```

```cpp
map<string, unsigned> products
{
	{"break", 30}, {"milk", 80}, {"apple", 60}
};

cout << "Products count: " << products.size() << endl; // Products count: 3
cout << "Products is empty: " << products.empty() << endl; // Products is empty: 0
```
