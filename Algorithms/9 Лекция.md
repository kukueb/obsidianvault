## Последовательные:
- `vector` - массив переменного размера, поддерживает доступ к произвольному элементу, обеспечивает добавление и удаление элементов с любым номером
- `deque` - поддерживает произвольный доступ к элементу, но добавление и удаление в начале или в конце контейнера
- `list` - двунаправленный список, обеспечивает добавление и удаление в начале и конце контейнера
- `forward_list` - однонаправленный список. Добавление и удаление в начале и конце
- `array`- массив фиксированного размера, произвольный доступ к элементу. Вставка и удаление не допускаются
- `string` - контейнер, аналогичный контейнеру `vector`, но состоящий из символов. 

## Ассоциативные контейнеры
обеспечивают быстрый доступ к данным по ключу. Построены на основе сбалансированных деревьев
- `map` - словарь
- `set` - множество

Кроме контейнеров существует специальный вид контейнера - адаптер. Он базируется на стандартном контейнере, но выполняет только функции, заложенные в структуру данных.

К адаптерам относят:
- `stack`
- `queue`
- `priority_queue` - очередь, но элементы отсортированы

# Итераторы
Для обращения к элементам контейнера используется итератор - объект языка программирования, который способен перебирать элементы контейнера без явного указания особенности структуры данных
`<контейнер><тип контейнера>::iterator <имя итератора>`

`vector<int>::iterator`

## Основные операции над итераторами
- `==` - равенство позиций (итераторов)
- `!=` - различие итераторов
- `++` - переход к следующему элементу контейнера
- `--` - переход к предыдущему элементу контейнера


Для того, чтобы получить значение элемента, на который указывает итератор, необходимо произвести операцию разыменования при помощи унарного оператора `*`
## Арифметические операции
```cpp
it += 5 // смещение на 5 ячеек вправо
```
`advance(итератор, value)`  - увеличение
Пример: 
```cpp
advance(it, 5)
```

```cpp
cout << *(it+5); cout << *it + 5;
```
Приоритеты!!

## Методы начала и конца контейнера
Метод `begin()` устанавливает итератора на начало контейнера (на первый элемент).
Метод `end()` устанавливает итератора на конец контейнера (одну ячейку после последней).

```cpp
vector v = {1,2,3,4,5};
for (vector::iterator it = v.begin(); it != v.end(); ++it) {
	cout << *it << " ";
}
cout << endl;
```

## Реверсивные итераторы
`rbegin()` - устанавливает итератор на последний элемент контейнер,
`rend()` - перед первым элементом

```cpp
vector v = {1,2,3,4};
for (vector<int>::reverse_iterator it = v.begin(); it != v.rend(); ++it) {
	cout << *it << " ";
}

// то же самое
for (vector<int>::iterator it = v.end()-1; it != v.begin(); --it) {
	cout << *it << " ";
}
```

## Константные итераторы
`cbegin()` - возвращает константный (только для чтения) итератор, представляющий начало элементов контейнера
`cend()` - возвращает константный (только для чтения) итератор, представляющий элемент, который находится после последнего элемента в контейнере

# Контейнер vector. Основные функции работы с vector
```cpp
#include <vector>
// 1. Создание пустого контейнера vector
// vector <тип_элемента> <имя>
vector <int> v1;
vector <double> v2;

// 2. Создание с заданным размером
// <тип_элемента> <имя>(размер)
vector<int> v1(10);

// 3. Размер + начальное значение
// <тип_элемента> <имя>(размер, значение)
vector<int> v1(10, 2);

// С различными значениями
vector<int> v1 = {1,2,3,4};

// 5. Создание контейнера и инициализация со значениями элементов данной стрктуры данных из полу-диапазона [fist, last)

int arr[7] = {15, 2, 19, -3, 28, 6, 8};
vector<int> v1(arr, arr + 7); // 15, 2, 19, -3, 28, 6

// 6. Создание контейнера с инициализацией его элементов значениями элементов другого однотипного контейнера
vector<int> v1;
// добавление элементов в v1 ...
vector<int> v2(v1);
```

## Метод `at`
Для обращения к отдельному элементу вектора можно использовать квадратные скобки или метод `at`, который обеспечивает безопасный доступ, проверяя границы диапазона вектора
```cpp
v[0] = 3;
v.at(1) = 4;
```

## Ввод значений вектора (`push_back()`)
Определим пустой вектор:
```cpp
vector<int> a;
```
Функция `push_back()` вставляет элемент в конец вектора.
```cpp
int main() {
	vector<int> a;
	int x;
	while (cin >> x) {
		a.push_back(x);
		print_vector(a);
	}
	return 0;
}
```

## Доступ к элементам вектора

1. Доступ к элементам вектора по значению
```cpp
vector v = {0,10,20,30,40,50};

for (auto i : v)
	cout << i << " ";
```
Аналогичное решение:
```cpp
vector v = {0,10,20,30,40,50};
for (int k = 0; k < v.size(); ++k) {
	int i = v[k];
	cout << i << " ";
}
```

2. Доступ к элементам вектора по константной ссылке
```cpp
vector v = {0,10,20,30,40,50};

for (const int &i: v) {
	cout << i << " ";
}
```

```cpp
vector v = {0,10,20,30,40,50};

for (auto &i: v) {
	cout << i << " ";
}
```

4. Доступ к элементам вектора с помощью прямого и обратного итератора
```cpp
for (vector<int>const_iterator it = v.begin(); it != v.end(); ++it)
	cout << *it << " ";
```

```cpp
for (vector<int>reverse_iterator it = v.rbegin(); it != v.rend(); ++it)
	cout << *it << " ";
```
