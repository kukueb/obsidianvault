## Моделирование очереди
```c++ 
void InQ(Queue &qe; Node x) {
  if (qe.k - qe.n >= max)
    cout << "переполнение очереди";
  else {
    if (qe.k == max) {
      for (int i = 0; i < max - qe.n; ++i)
        qe.q[i] = qe.q[i + qe.n - 1];
      qe.k = max - qe.n + 1;
      qe.n = 0;
    }
    qe.k++;
    qe.q[qe.k] = x;
  }
}
```

Подобная операция называется массовой т.к. приводит к перемещению большого
числа элементов структуры данных

## Кольцевой буфер

Для того чтобы избежать массовых операций линейную структуру данных
можно модифицировать в кольцевой буфер

```c++
#define max 10
    struct Queue {
  int n, k, kol;
  Node q[max];
};
Queue qe;
void InitQ(Queue &qe) {
  qe.n = 0;
  qe.k = 0;
  qe.kol = 0;
}
``` 
Введем дополнительную функцию `Sled(int nom, int m)`(следующий)
параметрами которой являются значения последнего элемента в очереди
- nom и m размер очереди(буфера)

```c++ 
int Sled(int nom, int m) {
  if (nom < m)
    return nom + 1;
  else
    return 0;
}
#Здесь m - максимальный номер кольцевого буфера;
void InQ(Queue &qe; Node x) {
  if (qe.kol == max)
    cout << "переполнение очереди";
  else {
    qe.k = Sled(qe.k, max);
    qe.q[qe.k] = x;
    qe.kol++;
  }
}
```

## Реализация 2х однотипных стеков Предположим что существуют 2 стека с
размерами max1 и max2 однако при заполнении стека данными общее
количество заполненных элементов равно n < max1 + max2
```c++ 
struct Stek {
  int top1, top2;
  Node s[max];
};
Stek st;
void InitStek(Stek &St) {
  St.top1 = 1;
  St.top2 = max;
}
```

```c++ 
void Push1(Stek &St; Node x) {
  if (St.top2 - St.top1 == 1)
    cout << "переполнение";
  else {
    St.top1++;
    St.s[St.top1] = x;
  }
}
void Push2(Stek &St; Node x) {
  if (St.top2 - St.top1 == 1)
    cout << "переполнение";
  else {
    St.top2--;
    St.s[St.top2] = x;
  }
}
```

```c++ 
void Pop1(Stek &St; Node & x) {
  if (Empty1(St))
    Cout << "стек 1 пуст";
  else {
    x = St.s[St.top1];
    --St.top1;
  }
}
void Pop1(Stek &St; Node & x) {
  if (Empty2(St))
    Cout << "стек 2 пуст";
  else {
    x = St.s[St.top2];
    St.top1++;
  }
}
```

## Последовательное и связное распределение памяти Совокупность блоков
памяти используемых для узлов информационной структуры называется пулом
свободной памяти

Если логически соседние узлы информационной структуры
располагаются в физически соседних ячейках памяти,
такое распределение называется последовательным Если физические ячейки не
рядом то распределение связное
