# Потоки ввода и вывода. Стандартные потоки в C++
**Поток** - последовательность символов, к которой можно получить доступ
В C++ существует 4 стандартных потока:
- `cin` - класс, связанные со стандартным вводом
- `cout` - стандартный вывод
- `cerr` - класс, связанный со стандартной ошибкой и обеспечивающий небуферизированный вывод
- `clog` - класс, связанный со стандартной ошибкой и обеспечивающий буферизированный вывод

При открытии файла можно использовать класс `ios`, атрибуты которого задают режимы чтения файла
- `in` - открыть файл в режиме чтения
- `out` - открыть файл в режиме записи
- `app` - открыть файл в режиме добавления
- `ate` - при открытии переместить указатель в конец файла
- `binary` - открыть файл в бинарном режиме
- `trunc` - удалить содержимое файла, если он уже существует.

Для основных потоков:
`ifstream` - `ios::in`
`ofstream` - `ios::out`
`fstream` - `ios::in / ios::out`

# Общие операции над файлами
В `<fstream>` определены несколько классов и подключены заголовочные файлы `<ifstream>` - файловый ввод и `<ofstream>` - файловый вывод

Для работы с файлами необходимо подключить один из классов `<fstream>`, `<ifstream>`, `<ofstream>`

Для того, чтобы создать текстовый файл необходимо 
1. создать объект класса `<ofstream>`,
2. связать объект класса с физическим файлом 
3. выполнить запись в файл
4. закрыть файл

1-2 Шаги
```cpp
ofstream fout; 
// Объект создан, но не связан с файлом, в котором нужно записать строку
fout.open("cppstudio.txt"); // связываем обхект с файлом
```
3 Шаг
```cpp
fout << "Работа с файлами в C++"; // запись строки в файл 
```
4 Шаг
```cpp
fout.close(); // закрываем файл
```

Итог - создан файл со строкой
Шаги 1 и 2 можно объединить, то есть в одной строке создать объект и связать его с файлом
```cpp
ofstream fout("cppstudio.txt"); // создаём объект класса ofstream и связываем его с файлом cppstudio.txt
```

Для чтения из файла необходимо:
1. создать обхект класса `ifstream` и связать его с физическим файлом, из которого будет выполняться считываение;
2. прочитать файл;
3. закрыть файл.

```cpp
#include <fstream>
#include <iostream>
#include <string>

using namespace std;
int main() {
  setlocale(LC_ALL, "rus"); // корректное отображение кириллицы
  string   s;
  ofstream fout("f.txt"); // создаём объект класса ofstream для записи и связываем его с файлом f.txt
  fout << "Работа с файлами в C++"; // запись строки в файл
  fout.close();                     // закрываем файл
  ifstream fin("f.txt"); // создаём объект класса ifstream для чтения и связываем его с файлом f.txt
  fin >> s; // считываение строки из файла до пробела
  cout << s << endl;
  getline(fin, s); // считали строку из файла до конца
  fin.close(); // закрываем файл
  cout << s << endl; // напечатали эту строку
  
  return 0;
}
```

# Проверка корректности открытия файла
Первый способ:
```cpp
ifstream file("d:\\1\\файл.txt");
if (!file) {
  cout << "Файл не открыт \n\n";
  return -1;
} else {
  cout << "Всё ОК! Файл открыт!\n\n";
  return 1;
}
```

Второй способ:
```cpp
if (file.is_open()) // вызов метода is_open()
  cout << "Всё ОК! Файл открыт!\n\n" << endl;
else {
  cout << "Файл не открыт!\n\n" << endl;
  return -1;
}
```

## Пример 2. Создать файл. Записать в новый файл только те строки, которые начинаются с цифры
```cpp
#include <fstream>
#include <iostream>
#include <string>

using namespace std;
int main() {
  setlocale(LC_ALL, "rus"); // корректное отображение кириллицы
  int n;
  string s;
  ofstream f("f.txt");
  cin >> n;
  for (int i = 0; i < n; ++i) {
    cin >> s;
    f << s << endl;
  }
  f.close();

  ifstream h("f.txt");
  ofstream g("g.txt");
  while (!h.eof()) {
    getline(h,s);
    if (s[0] >= '0' && s[0] <= '9')
      g << s << endl;
  }

  h.close();
  g.close();

  return 0;
}
```

# Файловый указатель. Прямой доступ к файлам
Файловый указатель содержит значение текущей позиции записи файла.
При открытии файла указатель устанавливается на первую запись со значением 0

Если к записям файла необходимо обращаться по порядку, файл называется последовательностью.
Однако существуют функции доступа к записи по указателю - прямой доступ
`seekg()` для ввода данных
`seekp()` для вывода данных

Параметры:
1. Смещение, на которое следует переместить файловый указатель (измеряется в байтах)
2. флаг `ios`, который обозначает позицию, относительно которой будет выполняться смещения
Возможное варианты флагов:
- `ios::beg` - начало файла;
- `ios::cur` - текущее положение;
- `ios::end` - конец файла.

```cpp
ifstream &seekg(смещение, позиция)
ofstream &seekp(смещение, позиция)
file.seekg(15, ios::cur); // вперед на 15 байт относительно текущего местоположения
file.seekg(-17, ios::cur); // назад на 17 байт
file.seekg(24, ios::beg); // вперед на 24 байт от начала файла
file.seekg(25) // вперед на 25 байт от начала файла
file.seekg(-27, ios::end); // к 27-му байту от конца файла
file.seekg(0, ios::beg); // перемещается в начало
file.seekg(0, ios::end); // перемещается в конец файла
```

`tellg()`, `tellp()` возвращают абсолютную позицию файлового указателя только от начала строки
```cpp
int main() {
	ifstream fin("smth.txt");
	fin.seekg(0, ios::endl);
	cout << fin.tellg(); // Вывели размер файла в байтах
}
```

# Абстрактные контейнерные классы
В C++ существует стандартная библиотека `stl`, содержащая шаблоны контейнерных классов и итераторов

Контейнерные классы реализуют практически все структуры данных
Контейнерные классы могут быть последовательными, обеспечивают хранение конечного количества однотипных объектов в виде непрерывной последовательности

Ассоциативные последовательные контейнеры делятся на связные списки и контейнеры произвольного доступа

Типы последовательных контейнеров
- `vector` - массив переменного размера
- `deque` - двусторонняя очередь
- `list` - двусвязный список
- `forward_list` - односвязные список
- `array` - массив фиксированного размера
- `string` - контейнер, аналогичный контейнеру `vector`, состоящий из сиволов.
