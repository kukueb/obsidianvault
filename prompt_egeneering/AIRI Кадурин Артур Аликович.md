# Графовые нейронные сети
### Алгоритм передачи сообщений
1. Каждая вершина и ребро имеют векторное состояние
2. сообщение - это функция от состояния вершины и ребра
3. Агрегация - это функция над полученными сообщениями
4. Каждый "слой" графовой сети состоит из этих функций
5. На выходе в каждой вершине формируются "предсказания"
## SchNet
Архитектура глубокой нейронной сети, специально разработанная для моделирования квантовых взаимодействий в молекулах и материалах. Она стала одной из новаторских моделей, которая позволила применять методы глубокого обучения к задачам квантовой химии с высокой точностью и скоростью
Суперсилы этого подхода:
1. Инвариантность
2. Обучение на силы
3. Множество улучшений
4. Он работает!
